Alarm 505

UF DEVIATION — 체적 편차 알람

이 알람은 유압 모듈(Hydraulic)과 보호 모듈(Protective)에서 각각 측정된 체중 감소량 사이에 불일치가 발생할 때 나타납니다. 두 값의 편차가 허용 기준을 초과하거나, 반복적인 확인 후에도 문제가 해결되지 않을 경우 알람이 발생합니다.
Check 1

측정값 비교

Total WL HYD(유압 모듈 누적 WL)와 Total WL PRO(보호 모듈 누적 WL)를 비교합니다.

두 값의 차이가 540g을 초과할 경우 알람이 발생합니다.

임계 기준
△ WL 차이
> 540g → 알람 발생
Check 2

유량계 노후화 및 드리프트

IEC Plus 구성 전용. 유량계의 노후화나 치료 중 발생하는 드리프트(변동)를 계산하여 체크합니다. 치료 시간에 따라 허용 임계치가 달라집니다.

치료 시간별 허용 임계치
< 4.5시간
→
200g 기준
≥ 4.5시간
→
매 시간마다 +50g 증가
Check 3

반복된 알람 확인

사용자가 알람 145 및 알람 379를 세 번 확인(Confirm)한 후에도 문제가 해결되지 않으면 알람 505가 생성됩니다.

알람 145 ×3 확인
알람 379 ×3 확인
→ 505 발생
Alarm 506

FLOWMETER ALIGNMENT FAILED — 유량계 정렬 실패

이 알람은 치료 시작 시, 매 시간마다, 또는 투석액 유량 변경 시 수행되는 유량계 타레이션(Taration) 과정에서 발생하며, WL 정확도에 영향을 미칩니다.
CCS 구성 — 유압 계수 (Kc)
10,000 ± 36
Phase 31 완료 기준 허용 범위
CCS 구성 — 보호 계수 (Kp)
10,000 ± 640
Phase 31 완료 기준 허용 범위
IEC Plus 구성 — 유압 및 보호 계수
10,000 ± 304
유압 계수(Kc)와 보호 계수(Kp) 동일 기준 적용
유량계 자체 문제: D1c/D2c, D1p/D2p 유량계의 하드웨어 결함 또는 노후화로 인한 정렬 실패
밸브 내부 누수: EVDS1, EVDS2, EV2 밸브의 내부 누수로 인해 캘리브레이션 중 유량이 불안정해지는 경우
투석액 흐름 불안정: 캘리브레이션 수행 중 투석액 공급 흐름이 불규칙하거나 일시적으로 중단된 경우
주의 (IEC Plus 구성): 밸브 누수 체크가 2회 연속 실패하면 (64) SAFE STATE ACTIVATED 알람이 발생할 수 있습니다. 블랙박스 분석을 통해 근본 원인을 신속히 확인하십시오.
Analysis Method

문제 분석 및 트러블슈팅 방법

그래프 분석

Mediartis Tech의 그래프 기능을 사용하여 Total WL HYD와 Total WL PRO의 거동을 시각적으로 확인합니다.

두 선이 겹쳐 보이더라도 스케일이 다를 수 있으므로 반드시 실제 수치를 비교하여 편차를 확인해야 합니다.

엑셀 데이터 분석 (Black Box)

블랙박스 세션을 엑셀로 내보낸 후, 별도의 열(Delta WL)을 만들어 두 값의 차이를 행마다 계산합니다.

치료 과정 중 어느 시점에서 오차가 점진적으로 증가했는지 파악할 수 있습니다.

알람 506 분석: 블랙박스 파일을 분석하면 문제가 유량계 자체의 결함 때문인지, 아니면 밸브 누수 때문인지 구분하는 데 도움이 됩니다. 두 원인은 대처 방법이 다르므로 반드시 원인을 먼저 특정한 후 조치하십시오.
Black Box Procedure

블랙박스 데이터에서 Delta WL 확인 절차

1

데이터 내보내기

분석하고자 하는 블랙박스 세션 데이터를 Excel 파일로 내보냅니다. Mediartis Tech 소프트웨어에서 해당 치료 세션을 선택하여 내보내기를 수행합니다.

2

새 열 생성

엑셀 파일에서 새로운 열을 추가하고 이름을 "Delta WL"로 지정합니다. 이 열이 두 모듈 간 편차를 실시간으로 추적하는 계산 열이 됩니다.

3

수식 입력

해당 열의 각 셀에 Total WL HYD와 Total WL PRO의 차이를 구하는 수식을 입력합니다.

예: =P332-Q332 (데이터 열 위치에 맞게 조정)

수식을 전체 행에 적용하면 모든 레코드에 대해 자동 계산됩니다.

4

데이터 분석 및 임계치 확인

계산된 Delta WL 값이 각 레코드에서 점진적인 오차(Progressive Error)가 어느 시점부터 커지는지 확인합니다.

알람 임계치
Delta WL > 540g
(505) UF DEVIATION 발생
목표
오차 증가 시점 파악
치료 중 어느 단계에서 발생했는지 특정
알람 505 연계: Delta WL 값이 540g 초과 시점을 확인하여 해당 치료 시간대의 유압/보호 모듈 동작 이상을 집중 점검하십시오.
알람 506 연계: 블랙박스 파일 분석으로 유량계 결함과 밸브 누수를 구분하고, 원인에 맞는 부품 교체 또는 서비스 조치를 진행하십시오.
