1개요

이 도구로 무엇을 할 수 있나요?

Artis BBOX Analysis Helper Web Edition은 기존 Excel 매크로(.xlsm) 없이 웹 브라우저만으로 Artis 투석기의 블랙박스 데이터(.dat)를 분석할 수 있는 도구입니다. 설치가 필요 없고, 파일은 외부 서버로 전송되지 않으며 브라우저 내에서만 처리됩니다.

드래그 & 드롭 분석

.dat 파일을 화면에 끌어다 놓으면 즉시 파싱되어 결과가 표시됩니다. 여러 파일을 동시에 불러올 수 있습니다.

알람 이력 조회

알람 코드, 발생 시각, 설명을 테이블로 확인할 수 있으며 타입·파일·키워드별 필터링이 가능합니다.

CSV 내보내기

파일별 또는 전체 데이터를 CSV로 내보내 Excel에서 추가 분석에 활용할 수 있습니다.

오프라인 동작

인터넷 연결 없이도 사용 가능합니다. 데이터는 브라우저 메모리에서만 처리되며 외부로 전송되지 않습니다.

지원 브라우저: Chrome, Edge, Firefox, Safari 최신 버전. Internet Explorer는 지원하지 않습니다. HTML 파일을 브라우저로 열어 바로 사용하세요.

2파일 불러오기

DAT 파일을 불러오는 방법

두 가지 방법으로 .dat 파일을 불러올 수 있습니다. 여러 파일을 한 번에 불러오는 것도 가능합니다.

파일 불러오기 화면

Drop .dat files here

Or click to browse  ·  Multiple files supported
All Artis BBOX formats (8.09 / 8.33 / 8.60 / 9.05)

A
드래그 & 드롭 영역 탐색기 또는 바탕화면에서 .dat 파일을 이 영역으로 끌어다 놓으면 자동으로 분석이 시작됩니다.
B
클릭하여 파일 선택 영역을 클릭하면 파일 선택 대화창이 열립니다. Ctrl(⌘)+클릭으로 여러 파일을 동시에 선택할 수 있습니다.
1

기기에서 DAT 파일 준비

Artis 투석기에서 USB 등으로 블랙박스 .dat 파일을 PC에 복사합니다. 파일명은 TalkFile_BB11_YYMMDD_HHMM_... 형태입니다.

2

파일을 드롭존에 끌어다 놓기

파일을 드래그해서 점선 박스 위에 놓으면 파란 테두리로 변하며, 놓는 순간 분석이 시작됩니다.

3

자동 포맷 감지 & 파싱

파일 크기를 기반으로 버전(8.09 / 8.33 / 8.60 / 9.05)을 자동 감지하고, 모든 레코드를 읽어 결과를 표시합니다.

파일 형식 오류: 지원되지 않는 파일 크기의 경우 "Unsupported format / file size" 오류가 표시됩니다. 파일이 손상되지 않았는지 확인하거나, 지원 담당자에게 문의하세요.

6CSV 내보내기

데이터 CSV로 내보내기

분석 결과를 CSV 파일로 저장해 Excel에서 추가 분석에 활용할 수 있습니다.

행별 CSV 버튼

테이블 각 행 우측의 ↓ CSV 버튼을 클릭하면 해당 파일 전체 데이터를 CSV로 내보냅니다.

Export All CSV

상단 Export All CSV 버튼을 클릭하면 불러온 모든 파일의 CSV를 순서대로 내보냅니다.

CSV 파일명: 원본 .dat 파일명에서 확장자를 제거하고 _1.csv를 붙인 이름으로 저장됩니다.
예: TalkFile_BB11_180326_2151.dat → TalkFile_BB11_180326_2151_1.csv

8지원 포맷

Artis BBOX 지원 버전 & 파일 구조

파일 크기를 기반으로 버전을 자동 감지합니다. 아래 4가지 포맷을 지원합니다.

지원 포맷 목록
버전 헤더 크기 레코드 크기 파일 크기 계산식
8.09 255 bytes 492 bytes / record 255 + (N × 492)
8.33 259 bytes 520 bytes / record 259 + (N × 520)
8.60 262 bytes 540 bytes / record 262 + (N × 540)
9.05 263 bytes 540 bytes / record 263 + (N × 540)

레코드 구조: 각 레코드의 첫 4바이트(uint32 LE)는 Unix 타임스탬프, 바이트 6–7은 타입 코드(uint16 LE), 바이트 8–11은 레코드 번호입니다. 알람(typeCode=0)의 경우 바이트 16–19에서 기술 코드(techCode)를 읽어 설명 메시지를 조회합니다.

알람 코드 지원 범위: 현재 코드 1–700번의 설명 메시지가 내장되어 있습니다. 범위 밖의 코드는 "Unknown Error"로 표시됩니다.
