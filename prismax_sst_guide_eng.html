Before You Begin

Date & Time Check

Before performing any SST, ensure that the date and time are correct on the device. SST results include timestamps for traceability.

How to Access

Press the SST menu item to display the SST Status screen. Select SST again to show the component test menu, then select the individual SST.

Results & Logs

The Report button captures a snapshot of all SST statuses. Results are retrievable via Service Logs in USB Import/Export tab in SysConfig.

Display SST
Verifies display and touchscreen meet specs: contrast, color resolution, pixel integrity, backlight gradient, status light color/intensity, and DCP function.
11 pages Operator-executed No tools required
Page Structure

Key Notes

Fluid & Air Systems 5 tests
4.6.3
ARPS SST
Verifies ARPS system meets specifications: pump, valves, pressure sensor assemblies, and tubing connections with no significant air leaks.
6 pages Operator interaction (pages 1–4) Tools required
Required Tools
SE6012 Pressure pod plug ×3 SE6020 Return port plug
Page Structure
Pages 1–2 — Apply specific pressures (0 & 450 mmHg) to internal ARPS
Pages 3–4 — Extend testing to include internal and external components
Page 5 — Sensor location tests; helps locate leaks if found
Page 6 — Summary of ARPS test results
Test Specs
Internal leakage ≤ 40 mmHg in 60 sec. External (valve to pods): leakage ≤ 10 mmHg in 300 sec. Location test: each sensor retains ~30 mmHg of pressure.

4.6.10
Motors SST
Verifies each motor meets specs: speed accuracy, odometer/encoder accuracy, start/stop response, motor currents, and correct location.
3 pages Mixed (auto + location) No tools required
Motors Covered
Blood pump, solution pumps (PBP, replacement, dialysate, effluent, AE).
Key Specs
Speed settle time < 5 sec. Settled speed within 2% of commanded. Std dev < 3 RPM. Speed within ±8 RPM at all times. Max current < 3.5 A. Stop time < 5 sec from 150 RPM.

4.6.22
RAM SST
Verifies main board RAM meets specs: data patterns can be written correctly over entire SRAM and DRAM memory spaces.
2 pages System-controlled No tools required
How It Works
SRAM used for startup; DRAM for normal operation. SST runs from one bank while testing the other. DRAM test requires running from SRAM (as part of boot code). Machine reboots and displays splash screen during RAM test.
4.6.23
TherMax SST
Verifies TherMax blood warmer device is functioning and can communicate with PrisMax. Covers thermistors, locking, LEDs, and heating plates.
4 pages Mixed (mostly operator) Tools required
Required Tools
SE7000 Bag Detect and Rear BLD Fixture
Page Structure
Page 1 — TherMax temperature and LED system tests
Page 2 — Display panel, power and safety tests
Page 3 — Locking mechanism and sensor tests (Optional)
Page 4 — Summary of all TherMax test results
Important Notes
Heater plates need time to cool before re-running page 2 tests. Cycle TherMax power when done — it will not communicate after safety system over-temperature SST until power-cycled. Timeout = 70 sec.
