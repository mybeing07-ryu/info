AK96 대비 변경사항 What's New vs. AK96
개요
💡
AK98은 AK96 대비 여러 구조적·기능적 변경이 이루어졌습니다. 아래 항목을 숙지하고 운영에 반영하십시오.

신형 컬러 터치스크린 적용.

Dialyzer holder arm이 재설계되어 Expansion chamber holder bracket에 통합됨.
💧 후면 소독액 인렛 고정화
영구적인 후면 소독액 인렛 튜브(Rear disinfectant inlet tube)가 추가됨.
※ 전면 Yellow chemical port 제거됨.

투석 준비 Dialysis Preparation
준비·장착
1
전원 켜기

조작
기기 전면 또는 패널의 전원(Power) 버튼을 눌러 기기를 기동한다.
2
약품 연결
Acid 약물과 Bicart를 연결한다.
조작
Acid 약물과 Bicart(중탄산염)를 기기에 연결한다.
3
투석액 선택 (필요시)
투석액 종류에 따라 적절한 투석액을 선택한다.
조건부
조작
필요한 경우 투석액 종류에 따라 화면에서 투석액을 선택한다.

혈액라인 장착 Blood Line Setup
준비·장착
1
동맥 라인 장착 (Artery Line)
Blood pump cover 열기 → 라인 장착 → cover 닫기
방향 주의
절차
  • Blood pump cover를 열고 blood line을 장착한 후 cover를 닫는다. (화살표 방향 주의)
  • Arterial chamber를 왼쪽 holder에 장착하고 방향을 확인한다.
  • 반대쪽 line을 clamp에 장착한다.
2
정맥 라인 장착 (Vein Line)
Venous chamber 장착 → clamp 연결 → Dialyzer 연결
절차
  • Venous chamber를 장착한다.
  • Venous clamp를 장착하고 saline drain bag을 연결한다.
  • Dialyzer를 연결한다.

프라이밍 Priming
준비·장착
1
Blood pump 라인 장착 확인
Blood pump 버튼을 눌러 라인 장착 여부 확인 후 펌프 정지
조작
Blood pump 버튼을 눌러 blood pump에 line이 올바르게 장착되는 것을 확인한 후 펌프를 정지한다.
2
센서 연결 및 클램프 닫기
Venous·Arterial 센서 연결 및 각 chamber 주입 라인 clamp close
Heparin 미연결 시 line clamp
절차
  • Venous sensor 및 Arterial sensor를 연결한다.
  • 각 chamber 위의 주입 라인 clamp를 close한다.
  • Heparin을 연결하지 않을 경우 line을 clamp한다.
3
Saline 연결
Arterial line에 saline을 연결한다.
조작
Arterial line에 saline(생리식염수)을 연결한다.
4
Prime 실행
Prime 버튼 → Manual 선택 → Blood pump 가동
자동 정지 있음
절차
  • Prime 버튼을 눌러 Manual 모드를 선택하고 확인 후 blood pump를 가동한다.
  • Arterial chamber를 채운 후 바로 세우고, dialyzer의 공기가 제거되면 venous chamber를 knob을 돌려 적절 부위까지 saline으로 채운다.
  • 설정 volume만큼 prime이 완료되면 펌프가 자동으로 정지된다.

재순환 및 설정 Recirculation & Setting
준비·장착
1
투석액 준비 확인 및 Fluid bypass
투석액 준비 완료 → fluid tube 연결 → Fluid bypass 버튼 누르기
절차
  • 투석액 준비가 완료되면 화면의 Fluid bypass 버튼이 깜박인다.
  • Fluid tube를 dialyzer에 연결하고 Fluid bypass 버튼을 누른다.
2
라인 연결 및 재순환 시작 (선택)
Artery·Vein 라인 연결 → Recirculation 및 Blood pump 가동
선택 사항
절차
  • Artery와 Vein line을 연결한다.
  • Blood pump 아래 infusion line saline을 연결하고 Fluid bypass clamp를 열어준다. (선택)
  • 화면의 Recirculation을 누르고 blood pump를 가동한다. (선택)
3
체중·시간·Heparin 설정
Time 버튼 → 체중·시간 확인 / Blood 버튼 → Heparin 설정
절차
  • Time 부분을 눌러 체중과 치료 시간을 확인·설정한다.
  • Blood 버튼을 눌러 Heparin을 설정한다.

환자 연결 Patient Connection
치료
⚠️
주의: 환자 연결 전 Blood line이 Priming detector에 올바르게 연결되어 있는지 반드시 확인하십시오.
1
사전 확인
Infusion line clamp 확인 및 Priming detector 연결 확인
필수 확인
확인 항목
  • Blood pump 아래 infusion line이 clamp된 것을 확인한다.
  • Blood line이 Priming detector에 올바르게 연결되어 있는지 확인한다.
2
환자 연결 시작
화면 "Connect patient" → Arterial line 환자 연결 → Venous line drain 연결
절차
  • 화면의 Connect patient를 누른다.
  • Arterial line을 환자에게 연결하고, Venous line을 drain에 연결한다.
  • Blood pump를 가동한다.
3
혈액 감지 후 정맥 연결 및 치료 시작
Priming detector 혈액 감지 → 펌프 자동 정지 → Venous line 환자 연결
자동 정지
절차
  • Priming detector에서 혈액을 감지하면 펌프가 자동으로 정지한다.
  • Venous line을 환자에게 연결한다.
  • 펌프를 가동하여 치료를 시작한다.

치료 시작 및 진행 Treatment Start & Progress
치료
1
Blood flow 설정
+/– 버튼으로 환자에 맞게 혈액 유량을 설정한다.
조작
+/– 버튼을 이용해 환자 처방에 맞게 혈액 유량(Blood flow rate)을 설정한다.
2
UF 버튼 활성화
UF 버튼이 깜박이면 눌러준다.
조작
Ultrafiltration(UF) 버튼이 깜박이면 눌러준다.
3
각 버튼 활성화 및 모니터링
Vein·Artery 안정 후 깜박이는 버튼 활성화 및 Venous chamber 수위 조절
절차
  • Vein과 Artery가 안정되면 깜박이는 각 버튼을 순서대로 눌러준다.
  • Venous chamber의 수위를 적절히 조절한다.
  • 화면의 남은 치료 시간이 감소하는 것을 확인한다.
  • 치료 중 blood pump 속도 조절 시 artery/vein 창이 깜박이면 해당 창을 눌러준다.

치료 종료 Treatment End
종료·소독
ℹ️
알람 코드 596 Treatment time expired가 발생하면 치료 시간이 종료된 것입니다. Confirm을 눌러 계속하거나 아래 절차를 따릅니다.
1
치료 종료 확인 및 Rinse back 시작
치료 시간 0 확인 → Rinse back 터치 → 시간 패드에 0 입력
절차
  • 치료 시간이 0이 아닌 경우: Rinse back을 터치하면 시간 패드가 나타남. 0을 입력한다.
  • 화면에 Rinse back이 한 번 더 나타나면 터치한다.
    (이후 투석으로 복귀 불가)
⚠️ Rinse back 진행 후에는 투석 상태로 돌아갈 수 없습니다.
2
Saline 연결 및 Rinse back 실행
Arterial line → saline 연결 → Blood pump 가동 → 자동 정지
자동 정지
절차
  • Arterial line을 saline에 연결하고 blood pump를 가동한다.
  • Rinse back volume이 완료되면 blood pump가 자동으로 정지한다.
3
환자 분리 및 정리
환자 분리 확인 → Disconnect patient → Blood line 분리 → Air detector 비활성화
환자 분리 필수 확인
절차
  • 반드시 환자와 분리가 된 것을 확인하고 Disconnect patient를 누른다.
  • Venous blood line을 Priming detector에서 분리한다.
  • Air detector를 비활성화한다.

투석기 & Bicart 비우기 Dialyzer & Bicart Empty
종료·소독
1
Dialyzer 방향 전환 및 Fluid tube 재연결
파란 Fluid tube 분리 후 기계 연결 → 투석기 붉은색 아래 방향 배치
방향 주의
절차
  • 파란색 fluid tube를 dialyzer에서 분리하여 기계에 연결한다.
  • 이때 dialyzer는 붉은색이 아래로 가도록 배치한다.
  • Dialyzer의 fluid part가 비워지면 붉은색 fluid tube를 기계에 장착한다.
2
Bicart Empty 실행
Suction tip 연결 → Bicart empty 버튼 → Bicart 상단 오픈(시간 단축)
시간 단축 팁
절차
  • Suction tip을 기계에 연결한다.
  • 화면의 Bicart empty를 누른다.
  • 소요 시간 단축을 위해 Bicart 상단 부분을 오픈한다.
  • 비워지면 Bicart cover를 닫고 소독을 진행한다.
  • Blood line을 기계에서 분리한다.
