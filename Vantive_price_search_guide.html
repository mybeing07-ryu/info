1화면 구성 개요

앱의 3가지 주요 영역

이 앱은 크게 헤더(검색 영역), 부품 카드 목록, 견적 요약 패널 세 구역으로 나뉩니다.

🔍

① 헤더 · 검색 영역

부품 이름이나 코드를 입력해 검색하고, 탭으로 콘텐츠 유형을 전환할 수 있습니다.

📦

② 부품 카드 목록

검색 결과로 나타난 부품별 카드입니다. 수량 조절과 즐겨찾기 추가가 가능합니다.

🧾

③ 견적 요약 패널

즐겨찾기에 담긴 부품의 합계, 서비스 요금, 할인을 계산해 최종 견적을 보여줍니다.

ℹ️

가격 기준 안내: 모든 가격은 2026년 4월 1일 기준이며, 환율에 따라 변동될 수 있습니다. 부가세(VAT) 별도 / 포함 금액이 함께 표시됩니다.

메모 & 출력 영역 미리보기

A

✅

PDF 저장 방법: 버튼 클릭 → 브라우저 인쇄 창 → 대상(프린터) 선택란에서 "PDF로 저장" 선택 → 저장 위치 지정 후 저장.

7이용 흐름

견적 작성 단계별 흐름

처음부터 견적서 출력까지 권장 순서입니다.

1

부품 검색

검색창에 부품 코드 또는 키워드를 입력합니다. 예: module, G5006802, blood pump

2

수량 조절 & 즐겨찾기 추가

카드의 −/+ 버튼으로 수량을 맞춘 뒤, ★ 버튼을 눌러 견적에 추가합니다. 여러 부품을 반복해서 추가하세요.

★이 노란색으로 바뀌면 추가 완료
3

서비스 요금 설정

Service Fee 체크박스를 켜고, 작업 시간(hr)을 입력합니다. 요금이 자동 계산됩니다.

시간당 150,000원 (VAT 별도)
4

할인율 적용

5% / 10% / 15% / 20% 버튼 중 하나를 선택하거나 직접 입력합니다. 최종 금액이 즉시 반영됩니다.

5

메모 입력 (선택)

고객사, 담당자, 작업 일정 등 특이사항을 메모란에 입력합니다.

6

견적서 출력 / PDF 저장

하단의 초록 버튼을 눌러 인쇄 또는 PDF로 저장합니다. 완료 후 "즐겨찾기 전체 삭제"로 초기화하면 다음 견적을 바로 시작할 수 있습니다.

새 견적 시작 전 반드시 즐겨찾기 초기화
